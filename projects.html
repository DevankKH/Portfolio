My Portfolio

Description

This project was created to serve as a portfolio for all my work. I will use this site to showcase my personal projects, as well as all the design work I have done. The website has a left panel with my personal details and a right panel with a navigation bar and where all the content will be shown.

The main sections include the Home tab for when the website first shows and it gives some information about myself. The second tab is the projects tab that shows you all my work created through coding, you can click the relevant buttons to open the details about a project and all of them can be found on GitHub. The third tab is all my design work I have done and they are sorted in an accordian format to store the information neater and it looks ordered.
Lastly there is a download CV tab to download a 1 page version of my CV.
