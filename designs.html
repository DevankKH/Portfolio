Project 2

The below SL project was designed to create a music quiz game where users would guess the artist of the song shown, users could only attempt each question once and needed to obtain a certain score to pass. If the correct answer was chosen then that choice would not be able to be picked again and the button would change to green and the score would increase. If incorrect then the button would be set to red and no points would be obtained. The states would be saved as you could come back to a question if you were not sure and it would skip already answered questions.
